# Yorui's Game of Life
A simulator that allows you to create circuit-like structures, but instead of electricity it uses colors :D


# CONTROLS

## W/A/S/W
Move

## Z/X/Mouse Wheel
Increase/Decrease Pixel Size (Zoom)

## Esc
Quit


## R
Clear

## 1/2/3
Selects Cells 1/2/3


## Space
Pause/Unpause

## C
Activate/Deactivate Selection Mode

## V
Activate/Deactivate Print Mode


## Left/Right Mouse (Drawing Mode)
Paint/Erase

## Left/Right Mouse (Selection Mode)
Paste/Select and Copy

## Left/Right Mouse (Print Mode)
Nothing (Yet)/Select and Print


# Cells
There are currently 2 new Cell Types, Cells 0 and 1 work the same as regular GoL, but the 2 new ones do new cool stuff >:)
